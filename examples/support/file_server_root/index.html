Applications

Aerys is an event-driven HTTP/1.1 web server written in PHP. Features include name-based virtual hosting, built-in SSL/TLS support, request logging, rate-limiting, request lifecycle event hooks for caching/customization and an army of gnomes to serve your every whim.

Static Files

Serve thousands of static files per second using Aerys's HTTP/1.1-compliant file server. Supports range requests, caching headers, streaming media, etc. Host static files on dedicated subdomains and make future scaling to CDNs and optimized servers painless.

Websockets

Host websocket endpoints from a dedicated domain (e.g. chat.mysite.com), layer them on top of your main application host (e.g. mysite.com/chat) or use Aerys's reverse proxy feature to overlay ws:// and wss:// (TLS!) endpoints on top of pre-existing applications!
